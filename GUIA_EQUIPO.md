# Guía Completa de Producción: FrameFlow

Este documento consolida TODA la información estratégica, operativa y de roles del sistema FrameFlow para ingeniería de crecimiento en YouTube.

---

## 📅 1. Hoy: Enfoque y Cadencia Diaria

### Propósito de cada Fase
- **Idea + Investigación:** Define el éxito de la semana. Un buen tema + keyword = video con tracción orgánica.
- **Guion + Gancho:** El guion es el esqueleto del video. Sin un buen guion, la edición no puede salvarlo.
- **Grabación:** Día de ejecución. Todo lo planificado se materializa hoy. Graba más de lo que necesitas.
- **Edición + Miniatura:** El editor trabaja en el primer corte. El creador prepara el packaging (miniatura + SEO).
- **Review + Cambios:** Día de pulir y aprobar. Máximo 1-2 rondas de cambios.
- **Publicación:** Las primeras 2 horas son CRÍTICAS. El algoritmo decide el alcance inicial aquí.
- **Métricas + Descanso:** Revisa métricas, responde comentarios y recarga para la próxima semana.

### Tiempos por Fase (Estimados)
| Fase | Quién | Tiempo |
| :--- | :--- | :--- |
| Idea + Investigación | Creador + Asistente | 1-2h |
| Títulos (Método Linden) | Creador | 30-60 min |
| Guion completo | Creador | 2-4h |
| Grabación | Creador | 2-6h |
| Edición (primer corte) | Editor | 6-12h |
| Review + cambios | Creador + Editor | 1-2h |
| Miniatura (3 versiones) | Asistente | 1-2h |
| SEO + Publicación | Creador + Asistente | 30-60 min |
| Post-pub (24h monitor) | Asistente | 1h activo |

## 👥 2. Roles y Compromisos

### Responsabilidades Detalladas

#### 👤 Creador (Creative Director)
- **Investigación:** temas únicos y validación de demanda.
- **Storytelling:** escritura de guiones usando la estructura Quería/Pero/Por lo tanto.
- **Ejecución:** grabación y energía frente a cámara.
- **SEO Titular:** 20+ variaciones de título antes de elegir.
- **Monetización:** estrategia de AdSense, productos y afiliados.

#### ✂️ Editor (Post-production)
- **Ritmo:** Estímulo visual/corte cada 10 segundos.
- **Estructura:** Proponer cortes que mejoren la narrativa.
- **Shorts:** Edición independiente de piezas verticales.
- **Gestión:** Mantener el orden de los proyectos en Drive.

#### 👤 Asistente (Growth/Operations)
- **Keyword Research:** buscar cola larga (long-tail) con cero competencia.
- **Diseño:** crear 3 versiones de miniatura (Principal + 2 backups).
- **Engagement:** responder TODOS los comentarios en las primeras 2h.
- **Redes:** publicar teasers y gestionar la comunidad.
- **Reporting:** reporte semanal de métricas (CTR, AVD, Subs).

### 🤝 El Handoff (Entregas)
| De | A | Gatillo | Contenido |
| :--- | :--- | :--- | :--- |
| Creador | Editor | Mover a "Edición" | Brutos en Drive, guion listo, gancho definido. |
| Editor | Creador | Primer corte listo | Video en Drive, notificación inmediata. |
| Creador | Editor | Feedback | Lista con timecodes (ej: "2:15 cortar silencio"). |
| Editor | Creador | Export final | Archivo en `05_Exports` y mover a "Publicación". |

### ⏱️ SLAs (Compromisos de Tiempo)
- **Brutos:** máximo 24h después de grabar.
- **Primer corte:** máximo 48h desde que se reciben los brutos.
- **Review:** máximo 12h para dar feedback.
- **Cambios finales:** máximo 24h (máximo 2 rondas).
- **Comentarios:** respuesta en las primeras 2 horas post-publicación.

## 📁 3. Organización y Comunicación

### Estructura de Google Drive
`[NombreCanal]/`
- `01_Guiones`: Scripts y escaletas.
- `02_Brutos`: Footage original.
- `03_Proyecto_Editor`: Archivos fuente de edición.
- `04_Miniaturas`: Diseños A, B y C.
- `05_Exports`: Videos finales.
- `06_Recortes_Shorts`: Material vertical.
- `07_Publicados`: Histórico.
*Formato: YYYY-MM-DD_TituloCorto*

### Canales de Comunicación
- **FrameFlow:** El cerebro. Aquí va el estado del video, feedback con timecodes, links y checklists.
- **WhatsApp/Telegram:** Solo notificaciones rápidas ("Brutos listos", "Aprobado"). **NUNCA** usar chat para organizar el trabajo detallado.

## ⚡ 4. Estrategia 10X (YouTube Growth)

### Ingeniería de Contenido y Retención
- **Fórmula de Retención:** Gancho (8s) -> Problema -> Desarrollo -> Clímax -> CTA.
- **Ingeniería del Clic:** Miniatura con Rostro + Texto (<4 palabras) + Objeto Visual.
- **Efecto Telaraña (Interlinking):** Usar comentario fijado con link a video relacionado + Pantalla final con el "mejor video para el espectador".
- **SEO Cola Larga:** 4+ palabras, menos volumen, CERO competencia. Buscar en autocompletado y Google Trends. Poner en título y 2 primeras líneas de descripción.

### Shorts: El Embudo
- **3 Pilares:** Gancho 1-3s, Formato repetible, Loopabilidad (inicio conecta con fin).
- **Funnel:** Shorts = Alcance (Top of Funnel). Largos = Monetización (Bottom). Cada Short canaliza a un largo.

### Monetización: 4 Fuentes
1. **AdSense:** Base (nichos con alto CPM: tecnología, finanzas).
2. **Afiliados:** 2-10x más que AdSense.
3. **Sponsors:** Cobro por CPM del canal.
4. **Producto Propio:** El mayor margen posible.

### 🚨 Protocolo de Emergencia
**Si CTR < 4% en las primeras 2 horas:**
1. Cambiar miniatura por el Backup B.
2. Esperar 1 hora. Si no mejora, cambiar título por un backup curioso.
3. Registrar en la tarjeta de FrameFlow qué cambio se hizo.
4. **Regla de Oro:** Nunca cambiar miniatura y título al mismo tiempo.
